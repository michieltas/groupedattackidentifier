Identifying grouped attacks

Context

Given a test stream of attacks of different types and attackers, the software automatically groups the attacks which belong together. A time gap of 1 minute is assumed to be a new attack.

Method 'public static void main(String[] args)' of class Main contains the streaming steps.

Executing the program

Method 1: Class Main.java can be run within an IDE, and produces print output to the console. For example, import the maven project into Eclipse, right-click class Main and choose 'Run as', choose Java application.

Method 2: Open a cmd or shell window, change directory to the top directory, where also the pom.xml is located, and type:
java -jar groupidentify_runnable.jar

Changing the test set

The list of attacks has for now been hard-coded and can be found in method generateDemoAttacks() from class AttacksGenerator.
